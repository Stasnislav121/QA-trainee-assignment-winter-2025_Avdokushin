Предусловия:
1. Отобран существующий продавец в любым количеством объявлений
2. Подготовлено тело запроса для добавления объявлений
{
    "name": "Телефон",
    "price": 85566,
    "sellerId": 345971 -- id продавца из шага 1
}

Шаги:
1.	Выполнить GET-запрос по адресу https://qa-internship.avito.com/api/1/:sellerID/item с "sellerID" из предусловий 
(например, 345971)
2. Выполнить POST-запрос по адресу https://qa-internship.avito.com/api/1/item с телом и с "sellerID" из предусловий:
{
    "name": "Телефон",
    "price": 85566,
    "sellerId": 345971 -- id продавца из шага 1
}
3. Повторно выполнить GET-запрос по адресу https://qa-internship.avito.com/api/1/:sellerID/item с "sellerID" из 
предусловий 
4. Сравнить ответы метода GET /:sellerID/item до и после добавления товара


Ожидаемый результат:
1.	Статус код-200
2.	В ответе метода GET /:sellerID/item количество товаров увеличилось на один
3. Добавленый товар расположен в конце массива ответа
4. Данные объекта товара соответствуют данным из тела запроса из предусловий
5. Структура товара соответствует формату 
{
